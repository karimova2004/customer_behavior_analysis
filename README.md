# Customer Shopping Behavior Analysis

## Overview
This project analyzes customer shopping behavior using Python, SQL, and Power BI. The project includes data cleaning, exploratory data analysis (EDA), SQL queries, interactive dashboard creation, and business insights.

## Dataset
- Customer shopping behavior dataset
- Cleaned and transformed using Python

## Tools & Technologies
- Python (Pandas, NumPy)
- SQL
- Power BI
- Google Colab
- Git & GitHub

## Project Workflow
- Data Loading
- Exploratory Data Analysis (EDA)
- Data Cleaning
- Feature Engineering
- SQL Analysis
- Power BI Dashboard
- Business Report
- Gamma Presentation

## Dashboard
The Power BI dashboard includes:
- Total Revenue
- Average Purchase Amount
- Revenue by Category
- Revenue by Gender
- Top Products
- Seasonal Sales
- Interactive Filters

## Results
The project provides insights into customer purchasing behavior, sales trends, product performance, and customer demographics to support business decision-making.

## How to Run
1. Clone this repository.
2. Open the Jupyter Notebook in Google Colab or Jupyter Notebook.
3. Run all cells.
4. Open the Power BI (.pbix) file to explore the dashboard.
